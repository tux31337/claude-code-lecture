# claude-code-lecture

> **"실리콘밸리 엔지니어의 Claude Code"** 강의 실습 워크북.

이 repo에는 학습자가 직접 채울 **빈 양식 파일(templates)** 만 들어 있습니다. 실습 가이드와 슬라이드는 강의 사이트에서 봅니다.

## 사용법

1. 우측 상단 **Fork** 버튼으로 본인 계정에 복제.
2. fork한 repo를 clone:
   ```bash
   git clone https://github.com/<your-username>/claude-code-lecture.git
   cd claude-code-lecture
   ```
3. 강의 사이트에서 챕터의 실습 가이드를 읽고, 같은 챕터의 `templates/<양식>.md`를 **직접 열어** 본인 답으로 채워 commit.
